# final-project

> Final Project for COMP 584

## DB Setup

``` bash
# run mongo locally on port 27017
mongod
```

## Server Setup
``` bash
# go to server folder
cd ./server

# install dependencies
npm install

# serve with client at localhost:1337
npm run start
```

## Client Setup

``` bash
# go to client folder
cd ./client

# install dependencies
npm install

# serve with client at localhost:8080
npm run start
```
